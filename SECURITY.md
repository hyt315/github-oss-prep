# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it via [GitHub Private Vulnerability Reporting](https://github.com/<owner>/<repo>/security/advisories/new) or email us at [security@example.com](mailto:security@example.com).

**Please do not report security vulnerabilities through public GitHub issues.**

We will respond within 5 business days and aim to resolve confirmed issues within 30 days.

## Supported Versions

As this is an AI Agent Skill project, only the latest version is actively supported.
